# Python Math Functions

This project contains basic mathematical functions implemented in Python:
- Exponentiation
- Square Root
- Factorial

---

## Functions Used

### 1. Exponentiation
Calculates power of a number.

Example:exponention(2, 3) → 8
---

### 2. Square Root
Finds square root of a number.

Example:

squareroot(16) → 4.0


---

### 3. Factorial
Calculates factorial of a number using recursion.

Example:

factorial(5) → 120


---

## How to Run

1. Run the Python file
2. Enter inputs when prompted
3. Get results for:
   - exponentiation
   - square root
   - factorial
